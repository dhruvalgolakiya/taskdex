Features

Everything you need to go mobile-first

Multi-agent management

Run and monitor multiple Codex agents simultaneously from a single mobile interface.

Bridge architecture

Lightweight relay server runs locally or on a VPS — your phone never talks directly to the API.

Live Activities

Track agent progress right from your iOS lock screen with real-time Live Activities.

File browser & Git

Browse project files, view diffs, and manage Git operations without leaving the app.

Offline queue

Commands queue while offline and sync automatically when connectivity is restored.

Voice input

Dictate prompts and instructions to your agents using built-in voice recognition.

How it works

Three pieces, one workflow

Pylon App

Send prompts, review output, and approve changes from your phone

Bridge Server

Lightweight relay that manages WebSocket connections and agent state

Codex API

OpenAI's coding agents execute tasks in your development environment

Built in the open

Pylon is fully open source. Inspect every line, self-host the bridge, and contribute features back to the community.
